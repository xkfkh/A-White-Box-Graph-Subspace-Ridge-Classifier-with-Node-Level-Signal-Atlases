# 05 src_v1 Layered Closed-form White-box Pipeline

Goal: build a modular closed-form white-box graph classifier.

Modules:
- data loading;
- Tikhonov smoothing;
- class PCA / SVD subspaces;
- response and impact computation;
- closed-form prediction.

Main lesson: this stage established the class-subspace view that later became the PCA residual branch.
